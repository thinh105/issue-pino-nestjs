## Description

Use to report a werid behaviour in the pino-nestjs pacakge: https://github.com/iamolegga/nestjs-pino/issues/1916

This project is generated by latest nest-cli with latest pino-nestjs package


## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode
$ npm run start:dev
```
